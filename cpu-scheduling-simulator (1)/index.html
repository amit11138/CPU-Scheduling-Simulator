Introduction to CPU Scheduling

CPU scheduling is the process by which a CPU decides which of the processes in the ready queue is to be allocated to the CPU. Below is the default set of processes and a Gantt chart illustrating the FCFS algorithm.

First Come, First Served (FCFS)

The first process to arrive is the first to be executed. Simple and fair, but can lead to long waiting times for short processes stuck behind long ones (the "convoy effect").

Gantt Chart

Performance Metrics

Shortest Job First (SJF)

Selects the process with the shortest burst time to execute next. Optimal in minimizing average waiting time, but requires knowing burst times in advance. Change burst times below and see the effect.

Gantt Chart

Performance Metrics

Priority Scheduling

Each process is assigned a priority, and the process with the highest priority (lower number) is executed first. Can lead to "starvation" for low-priority processes.
